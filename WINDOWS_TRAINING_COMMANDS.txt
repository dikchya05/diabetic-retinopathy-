================================================================================
                              NOTE ABOUT BACKSLASHES
================================================================================

DON'T use backslashes (\) for line continuation in Windows CMD!

WRONG (Linux style):
python train_model.py --labels-csv ml/data/train.csv \
                      --img-dir ml/data/train_images \
                      --epochs 5

RIGHT (Windows - all on one line):
python train_model.py --labels-csv ml/data/train.csv --img-dir ml/data/train_images --epochs 5

OR use ^ in CMD:
python train_model.py --labels-csv ml/data/train.csv ^
                      --img-dir ml/data/train_images ^
                      --epochs 5

OR use ` in PowerShell:
python train_model.py --labels-csv ml/data/train.csv `
                      --img-dir ml/data/train_images `
                      --epochs 5
